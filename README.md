# gshop
    一个外卖平台的项目
# 项目相关
    1.下载依赖
    yarn install
    2.启动项目
    yarn start
    3.一般访问
    http://localhost:8080/#/msite
# mongodb安装
    启动
    C:\Program Files\MongoDB\Server\3.2\bin
    _> mongod
# 服务端项目下载 
    git clone git@github.com:liuer1211/vue-serve.git
    1.下载依赖
    yarn install
    2.启动项目
    yarn start
